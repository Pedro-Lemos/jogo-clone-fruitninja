# Stack Slicer

**Stack Slicer** é um jogo web estilo "Fruit Ninja" criado para fins acadêmicos. 

Este repositório foi desenvolvido para uma aula da faculdade, especificamente para a disciplina de **Inteligência Artificial**. 
O diferencial do projeto é que ele foi inteiramente concebido e programado utilizando **Context Coding** (anteriormente conhecido como *Vibe Coding*), testando as capacidades de IA agentiva na criação de software do zero a partir de prompts naturais.

---

O jogo é uma brincadeira com tecnologias do desenvolvimento de software: o objetivo é cortar (fazer um "slice") nas principais linguagens de programação para marcar pontos, enquanto desvia e evita cortar a bomba: o **Python**! 🍎💥🐍
